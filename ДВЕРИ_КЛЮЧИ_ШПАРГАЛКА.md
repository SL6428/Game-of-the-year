## 🎮 Логика работы

### Ключ:
```
Подойти → [E] Подобрать → Нажать E → 🔑 Ключ подобран (2 сек) → Ключ исчезает
```

### Дверь без ключа:
```
Подойти → [E] Открыть → Нажать E → Дверь открывается (1.5 сек)
```

### Дверь с ключом (нет ключа):
```
Подойти → [E] Открыть → Нажать E → Требуется ключ (2 сек) → Можно попробовать снова
```

### Дверь с ключом (есть ключ):
```
Подойти → [E] Открыть → Нажать E → Дверь открывается (1.5 сек)
```

## ⚠️ Важно!

1. **Prompt Panel и Message Panel - РАЗНЫЕ объекты!**
2. **Key Name у ключа = Required Key Name у двери**
3. **Все UI панели скрыты по умолчанию**
4. **У игрока должен быть тег "Player"**
5. **На сцене должен быть GameManager**

---

## 🐛 Troubleshooting

| Проблема | Решение |
|----------|---------|
| Сообщение не появляется | Проверь что назначены Message Panel и Message Text |
| Подсказка не исчезает | Убедись что Prompt и Message - разные панели |
| Дверь не открывается | Проверь Requires Key и наличие ключа |
| Ключ не подбирается | Проверь что KeyPickup не isPickedUp |
| Ошибка "Player not found" | Добавь тег "Player" игроку или назначи в GameManager |
