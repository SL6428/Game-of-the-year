# 🚪 Система дверей и ключей

## 📋 Созданные скрипты:

| Файл | Назначение |
|------|-----------|
| `SimpleInventory.cs` | Хранение ключей (singleton) |
| `KeyPickup.cs` | Ключ на сцене (подбирается по E) |
| `Door.cs` | Дверь с анимацией (открывается по E) |

---

## 🛠 Настройка в Unity:

### 1️⃣ Создать ключ на сцене:

```
1. GameObject → 3D Object (модель ключа)
2. Add Component → KeyPickup
3. Настроить в инспекторе:
   - Key Name: "UpperHallKey" (уникальное имя)
   - Pickup Radius: 2
   - Player: перетащи игрока
   - Destroy After Pickup: ✅ (исчезает после подбора)
4. (Опционально) Назначь UI панели для сообщения "Подобран ключ"
```

### 2️⃣ Создать дверь БЕЗ ключа:

```
1. GameObject → 3D Object (модель двери)
2. Add Component → Door
3. Настроить в инспекторе:
   - Requires Key: ❌ (не нужен ключ)
   - Interaction Radius: 2
   - Rotation Axis: (0, 1, 0) - вращение вокруг Y
   - Open Angle: 90 - на сколько градусов открывается
   - Open Duration: 1.5 - скорость открытия
4. (Опционально) Назначь UI панели:
   - Prompt Panel: "[E] Открыть"
5. Важно: Поставь дверь в закрытое положение перед запуском!
```

### 3️⃣ Создать дверь С ключом:

```
1. GameObject → 3D Object (модель двери)
2. Add Component → Door
3. Настроить в инспекторе:
   - Requires Key: ✅ (нужен ключ)
   - Required Key Name: "UpperHallKey" (должно совпадать с KeyPickup!)
   - Key Missing Message: "Требуется ключ от верхнего зала"
   - Rotation Axis: (0, 1, 0)
   - Open Angle: 90
   - Open Duration: 1.5
4. Назначь UI панели:
   - Prompt Panel: "[E] Открыть"
   - Message Panel: панель для сообщения "Требуется ключ..."
   - Message Text: текст внутри панели
```

## 🎮 Как работает:

### Ключ:
| Действие | Результат |
|----------|-----------|
| **Подойти к ключу** | В радиусе 2 единицы |
| **Нажать E** | Ключ подобран, сообщение "Подобран ключ" |
| **Ключ исчезает** | Если Destroy After Pickup = ✅ |

### Дверь без ключа:
| Действие | Результат |
|----------|-----------|
| **Подойти** | Появляется "[E] Открыть" |
| **Нажать E** | Дверь плавно открывается (анимация 1.5 сек) |
| **После открытия** | Больше нельзя взаимодействовать |

### Дверь с ключом:
| Действие | Результат |
|----------|-----------|
| **Подойти без ключа** | Появляется "[E] Открыть" |
| **Нажать E (нет ключа)** | Сообщение "Требуется ключ от верхнего зала" |
| **Подойти с ключом** | Появляется "[E] Открыть" |
| **Нажать E (есть ключ)** | Дверь открывается |

---

## 🔧 Настройка анимации двери:

### Rotation Axis (ось вращения):
- **(0, 1, 0)** - вращение вокруг вертикальной оси (обычно)
- **(1, 0, 0)** - вращение вокруг X (редко)
- **(0, 0, 1)** - вращение вокруг Z (редко)

### Open Angle (угол открытия):
- **90** - дверь открывается на 90° (вправо)
- **-90** - дверь открывается на 90° (влево)
- **180** - дверь открывается на 180°

### Важно:
- Дверь **запоминает свой начальный поворот** при старте
- Открывается **относительно текущего положения** (работает под любым углом!)
- Не важно если дверь стоит под -453.12° - она откроется правильно!

- Скрыть по умолчанию

## 📝 Пример сцены:

```
Сцена:
├─ Player (тег: Player)
├─ Key (KeyPickup)
│   └─ Key Name: "UpperHallKey"
├─ Door1 (Door)
│   └─ Requires Key: ❌
└─ Door2 (Door)
    └─ Requires Key: ✅
    └─ Required Key Name: "UpperHallKey"
```

**Порядок действий:**
1. Подойти к Key → нажать E → ключ подобран
2. Подойти к Door1 → нажать E → открывается (без ключа)
3. Подойти к Door2 → нажать E → открывается (нужен ключ, он подобран)
