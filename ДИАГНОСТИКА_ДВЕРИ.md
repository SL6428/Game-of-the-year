```
[Door] Открываю дверь!

```

```

```

## Как диагностировать проблему

### 1️⃣ Откройте консоль Unity
```
Window → General → Console (или Ctrl+Shift+C)
```

### 2️⃣ Подойдите к двери в игре

**Если видите:**
```
[Door] Игрок вошёл в радиус
```
✅ Отлично! Дверь видит игрока.

**Если видите:**
```
[Door] playerTransform = null! Не могу найти игрока.
```
❌ Проблема: не найден игрок.

**Решение:**
1. Убедитесь что у игрока есть тег "Player" (точно написано, case-sensitive!)
2. Или перетащите игрока в GameManager → Player
3. Или создайте GameManager на сцене

---

### 3️⃣ Нажмите E рядом с дверью

**Если видите:**
```
[Door] E нажат! distance=1.50, interactionRadius=2.00
[Door] TryOpenDoor вызван | isOpen=False, requiresKey=False
[Door] Открываю дверь!
[Door] Дверь 'Door' открывается
```
✅ Отлично! Дверь пытается открыться.

**Если видите:**
```
[Door] E нажат! distance=1.50, interactionRadius=2.00
[Door] TryOpenDoor вызван | isOpen=False, requiresKey=False
[Door] Открываю дверь!
... и тишина
```
❌ Проблема: анимация не работает.

**Решение:**
- Проверьте что Open Duration > 0 (рекомендуется 1.5)
- Проверьте что Open Angle != 0 (рекомендуется 90)

---

### 4️⃣ Дверь не реагирует на E

**Если НЕ видите:**
```
[Door] E нажат! ...
```

**Возможные причины:**
1. ❌ Вы не в радиусе двери (distance > interactionRadius)
2. ❌ Игра на паузе (Time.timeScale = 0)
3. ❌ Input не работает (проверьте другие клавиши)

**Решение:**
1. Подойдите ближе к двери (смотрите расстояние в логах)
2. Нажмите Escape чтобы снять паузу
3. Проверьте что GameManager не на паузе

---

## Частые проблемы

### ❌ Проблема: "playerTransform = null"

**Причина:** Не найден игрок

**Решение:**
```
1. Выберите игрока в сцене
2. В инспекторе: Tag → Player
   (если нет тега Player: Tags & Layers → Add "Player")
3. Или: GameManager → Player → перетащите игрока
```

---

### ❌ Проблема: Подсказка появляется но E не работает

**Причина:** Расстояние больше чем interactionRadius

**Решение:**
```
1. Выберите дверь
2. Interaction Radius: увеличьте (2 → 3 → 4)
3. Или подойдите ближе
```

---

### ❌ Проблема: Дверь "открывается" но не двигается

**Причина:** Open Angle = 0 или ось вращения неправильная

**Решение:**
```
1. Выберите дверь
2. Rotation Axis: (0, 1, 0) - для вращения вокруг Y
3. Open Angle: 90
4. Open Duration: 1.5
```

---

### ❌ Проблема: Игра на паузе

**Причина:** Time.timeScale = 0 блокирует Time.deltaTime

**Решение:**
```
1. Нажмите Escape (снимает паузу)
2. Или проверьте GameManager → isPaused = false
```

**ИСПРАВЛЕНО:** Теперь используется `Time.unscaledDeltaTime` чтобы анимация работала даже на паузе! ✅
