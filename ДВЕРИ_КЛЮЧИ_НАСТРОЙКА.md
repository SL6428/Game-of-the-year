# 🔑 Двери и ключи - Настройка

## 📋 Скрипты:
- `SimpleInventory.cs` - хранит ключи
- `UIMessageManager.cs` - показывает сообщения
- `KeyPickup.cs` - ключ на сцене
- `Door.cs` - дверь с анимацией

---

## 🛠 Настройка:

### 1️⃣ Ключ (KeyPickup):
```
1. Создай объект ключа
2. Add Component → KeyPickup
3. Настрой:
   - Key Name: "UpperHallKey"
   - Pickup Radius: 2
   - Prompt Panel: твоя панель "[E] Подобрать"
4. Скрой Prompt Panel по умолчанию!
```

### 2️⃣ Дверь (Door):
```
1. Создай объект двери
2. Add Component → Door
3. Настрой:
   - Requires Key: ✅ или ❌
   - Required Key Name: "UpperHallKey" (если нужен ключ)
   - Key Missing Message: "Требуется ключ от верхнего зала"
   - Prompt Panel: твоя панель "[E] Открыть"
   - Rotation Axis: (0, 1, 0)
   - Open Angle: 90
4. Скрой Prompt Panel по умолчанию!
```

### 3️⃣ UIMessageManager (автоматически):
Не нужно создавать - скрипты создадут его сами!

---

## 🎮 Как работает:

### Ключ:
- Подходишь → появляется "[E] Подобрать"
- Нажимаешь E → сообщение "🔑 Подобран ключ: UpperHallKey" (2 сек)
- Ключ исчезает

### Дверь без ключа:
- Подходишь → "[E] Открыть"
- Нажимаешь E → дверь открывается

### Дверь с ключом (нет ключа):
- Подходишь → "[E] Открыть"
- Нажимаешь E → сообщение "Требуется ключ от верхнего зала" (2 сек)

### Дверь с ключом (есть ключ):
- Подходишь → "[E] Открыть"
- Нажимаешь E → дверь открывается
