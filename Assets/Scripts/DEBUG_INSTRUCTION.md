# 🔧 Инструкция по отладке попаданий

## ✅ Что исправлено:

### Главная проблема:
**Флаг `isAttacking` не устанавливался в `true`** при атаке, поэтому `OnTriggerEnter` сразу отменялся.

**Теперь:**
- `EnableHitbox()` устанавливает `isAttacking = true`
- Добавлена подробная отладка с эмодзи для каждого этапа
- Видно все проверки и причины отмены

---

## 🧪 Пошаговая проверка:

### Шаг 1: Повесь скрипт отладки на врага
```
1. Выбери врага в сцене
2. Add Component → EnemyColliderDebug
3. Запусти игру
```

В консоли увидишь:
```
=== [DEBUG] Враг: Enemy ===
[DEBUG] Тег врага: Enemy
[DEBUG] Сравнение с 'Enemy': True
[DEBUG] Коллайдеров на враге: 1
  - BoxCollider: isTrigger=False, enabled=True
[DEBUG] Health найден: HP=50/50
```

**✅ Если всё верно** — переходи дальше  
**❌ Если тег не Enemy** — назначь тег  
**❌ Если Health не найден** — добавь компонент Health

---

### Шаг 2: Запусти игру и атакуй врага

Смотри консоль. Должно быть:

```

[TRIGGER] === ВХОД В ТРИГГЕР ===
[TRIGGER] Объект: Enemy, Тег: Enemy

```

### Шаг 3: Если нет [TRIGGER] === ВХОД В ТРИГГЕР ===

**Проблема:** Коллайдер оружия не пересекается с коллайдером врага

**Причины:**
1. **Меч проходит "сквозь" врага** — слишком маленький коллайдер на оружии
2. **Неправильная позиция хитбокса** — смещён относительно модели меча
3. **Слои физики** — оружие и враг на слоях которые не взаимодействуют

**Решение:**

#### 1. Проверь коллайдер оружия:
```
1. Найди объект оружия на игроке
2. Выбери коллайдер на оружии
3. Включи "Is Trigger" (должен быть true)
4. Проверь размер коллайдера (должен покрывать лезвие меча)
```

#### 2. Проверь слои физики:
```
1. Edit → Project Settings → Physics
2. Найди "Layer Collision Matrix"
3. Убедись что слой оружия пересекается со слоем "Enemy"
```

#### 3. Визуальная проверка:
```
1. Запусти игру
2. Поставь на паузу во время атаки
3. Посмотри в сцену (Scene view)
4. Видишь ли зелёный проводфрейм коллайдера оружия?
5. Пересекается ли он с коллайдером врага?
```

---

### Шаг 4: Если есть [TRIGGER] но урон не наносится

Смотри почему отменилось:

```
[TRIGGER] ❌ Отмена: canDamage = false
→ EnableHitbox() не вызывается или вызывается слишком поздно

[TRIGGER] ❌ Отмена: isAttacking = false  
→ Анимация не вызывает EnableHitbox() в нужный момент

[TRIGGER] ❌ Объект не с тегом Enemy: Player
→ У врага не назначен тег "Enemy"

[HEALTH] ❌ Health не найден на Enemy
→ На враге нет компонента Health
```
