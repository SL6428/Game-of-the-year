## 🔧 Необходимая настройка в Unity Editor

### 1. Назначить тег "Player" игроку

1. Откройте сцену `Assets/Scenes/Gail map (first map).unity`
2. Найдите объект игрока
3. В инспекторе нажмите на тег (верхний правый угол)
4. Выберите **"Player"** из списка
5. Сохраните сцену (Ctrl+S)

### 2. Добавить компонент PlayerRegeneration на игрока

1. Выберите объект игрока
2. Нажмите **Add Component**
3. Найдите **"PlayerRegeneration"**
4. Настройте параметры (по умолчанию уже настроено):
   - Max Charges: 5
   - Recharge Time: 15
   - Heal Amount: 30

### 3. Настроить Weapon на игроке

1. Найдите объект оружия на игроке
2. Убедитесь что есть компонент **Weapon**
3. Проверьте настройки:
   - Damage: (ваше значение)
   - Enemy Layers: слой "Enemy"

### 4. Создать префабы (по желанию)

#### Префаб игрока:
1. Настройте игрока в сцене (тег "Player", все компоненты)
2. Перетащите объект из сцены в папку `Assets/Prefabs/`
3. Назовите `Player.prefab`

#### Префаб врага:
1. Настройте врага в сцене
2. Убедитесь что есть компоненты:
   - **EnemyAI**
   - **Health**
   - **HealthUIEnemy**
   - **NavMeshAgent**
3. Перетащите в `Assets/Prefabs/`
4. Назовите `Enemy.prefab`

#### Префаб оружия:
1. Найдите объект оружия
2. Перетащите в `Assets/Prefabs/`
3. Назовите `Weapon.prefab`

### 5. Настроить врагов

Для каждого врага в сцене:

1. Выберите врага
2. В инспекторе найдите компонент **EnemyAI**
3. Настройте параметры:
   - **Damage:** урон который наносит враг (по умолчанию 10)
   - **Attack Range:** дистанция атаки (по умолчанию 2)
   - **Attack Cooldown:** задержка между атаками (по умолчанию 1.5)
   - **Detection Range:** дистанция обнаружения игрока
   - **Patrol Points:** точки патрулирования (создайте пустые объекты и назначьте)

## 🎯 Как работает боевая система

### Бой игрока:
1. Нажмите **ЛКМ** → Атака оружием
2. Анимация вызывает `EnableHitbox()` в момент удара
3. При попадании во врага наносится урон
4. Хитбокс автоматически отключается после попадания

### Лечение игрока:
1. Нажмите **H** → Начинается анимация лечения
2. После анимации (1.5 сек) используется 1 заряд
3. Здоровье восстанавливается на 30 единиц
4. Каждый заряд перезаряжается 15 секунд
5. Максимум 5 зарядов

### Бой врага:
1. Враг обнаруживает игрока в радиусе `detectionRange`
2. Преследует игрока если видит
3. На дистанции `attackRange` начинает атаку
4. Атака наносит урон каждые `attackCooldown` секунд

### Смерть:
- При HP ≤ 0 вызывается событие смерти
- Враг: отключаются коллайдеры и рендер, уничтожение через 1 сек
- Игрок: (реализуйте свою логику)

## ⚠️ Возможные проблемы и решения

### Ошибка: "CameraPivot не найден"
**Решение:** Убедитесь что на сцене есть объект с компонентом CameraPivot

### Ошибка: "Игрок не найден для атаки"
**Решение:** 
1. Проверьте что у игрока есть тег "Player"
2. Проверьте что EnemyAI ссылается на игрока

### Лечение не работает
**Решение:**
1. Проверьте что на игроке есть компонент PlayerRegeneration
2. Проверьте количество доступных зарядов
3. Проверьте что HP не полное

### Оружие не наносит урон
**Решение:**
1. Проверьте что у врага есть тег "Enemy"
2. Проверьте что у врага есть компонент Health
3. Проверьте что коллайдер оружия — триггер
4. Проверьте логи в консоли
