# ♻️ Система респауна врагов

## ✅ Что добавлено:

### Враг теперь возрождается через 5 секунд после смерти!

**Как работает:**
1. Игрок убивает врага
2. Враг "умирает" (скрывается, HP = 0)
3. Через 5 секунд:
   - HP восстанавливается
   - Враг появляется на исходной позиции
   - Начинает патрулирование заново

---

## ⚙️ Настройка (в Inspector):

На выбранный объект врага, найди компонент **EnemyAI**:

```
Death:
  Death Effect: [эффект смерти, если есть]
  Disappear Delay: 1          // Задержка перед исчезновением
  Respawn Time: 5             // ⏰ Время до возрождения (сек)
  Respawn Enabled: ☑ True     // ✅ Включить респаун
```

### Отключить респаун:
Поставь `Respawn Enabled: False` — враг будет уничтожен навсегда

## 📊 Логика работы:

```
┌─────────────────────────────────────────────────────────┐
│                    СМЕРТЬ ВРАГА                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. Health.TakeDamage() → HP = 0                       │
│     ↓                                                   │
│  2. Health.Die() → вызывает событие OnDeath            │
│     ↓                                                   │
│  3. EnemyAI.OnDeath() → Die()                          │
│     ↓                                                   │
│  4. Скрываем врага (рендеры off, коллайдеры off)      │
│     ↓                                                   │
│  5. Invoke(Respawn, 5) ← таймер на 5 сек              │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│                   ВОЗРОЖДЕНИЕ                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. Respawn() через 5 сек                              │
│     ↓                                                   │
│  2. health.ResetHealth() → HP = 100%                   │
│     ↓                                                   │
│  3. Включаем рендеры и коллайдеры                      │
│     ↓                                                   │
│  4. transform.position = spawnPosition                 │
│     ↓                                                   │
│  5. agent.Warp() + agent.enabled = true                │
│     ↓                                                   │
│  6. isDead = false, Patrol state                       │
│     ↓                                                   │
│  ✅ Враг снова жив и патрулирует!                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 🎮 Геймплей:

### Теперь можно:
- ✅ Убить врага → он исчезнет через 1 сек
- ✅ Подождать 5 сек → он появится снова
- ✅ Бесконечный цикл "смерть-возрождение"

### Для постоянного уничтожения:
1. Выбери врага
2. В EnemyAI поставь `Respawn Enabled: False`
3. Или уничтожь врага через скрипт

---

## 🔧 Если что-то не работает:

### Враг не появляется:
- Проверь `Respawn Enabled: True`
- Проверь консоль на ошибки
- Убедись что враг не уничтожен (смотрел в иерархии?)

### Враг появляется но не двигается:
- Проверь что NavMesh запечён
- Проверь что `agent.Warp()` сработал (смотри логи)

### Враг появляется в воздухе:
- Проверь что spawnPosition на земле
- Убедись что NavMesh есть в точке респауна

---

## 💡 Советы:

### Разные типы респауна:

**Быстрый респаун (3 сек):**
```
Respawn Time: 3
```

**Медленный респаун (10 сек):**
```
Respawn Time: 10
```

**Без респауна (уничтожение):**
```
Respawn Enabled: False
```

### Для боссов:
```
Respawn Enabled: False  // Босс не возрождается
```

### Для обычных врагов:
```
Respawn Time: 5         // Стандартное время
```
