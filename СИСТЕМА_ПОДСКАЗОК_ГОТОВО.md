# 🎮 Система подсказок в стиле Dark Souls 3 - Готово!

## ✅ Что реализовано:

### Файлы скриптов:
1. **`InteractionManager.cs`** - Singleton менеджер (один на сцену)
2. **`InteractionUI.cs`** - Управление UI (prompt + popup)
3. **`InteractionUISetup.cs`** - Авто-создание UI при старте
4. **`InteractableObject.cs`** - Базовый класс для всех интерактивных объектов
5. **`Signpost.cs`** - Конкретный скрипт для подсказок/знаков

---

## 🚀 Как настроить в Unity:

### 1️⃣ Создать менеджер:
```
1. В Unity: GameObject → Create Empty
2. Переименовать в "InteractionManager"
3. Add Component → InteractionManager
```

### 2️⃣ Создать подсказку:
```
1. GameObject → 3D Object → Cube (или модель)
2. Переименовать в "Signpost"
3. Add Component → Signpost
4. Настроить в инспекторе:
   - Sign Title: "Заголовок подсказки"
   - Sign Text: "Текст подсказки..."
   - Interaction Radius: 2-3
   - Player: перетащи игрока
```

### 3️⃣ Убедись что у игрока тег "Player":
```
1. Выбери объект игрока
2. В инспекторе: Tag → Player
```

---

## 🎮 Как работает:

1. **Подходишь к объекту** → внизу экрана появляется `[Y] Прочитать`
2. **Нажимаешь Y** → открывается окно с текстом
3. **Нажимаешь B** → окно закрывается
4. **Выходишь из радиуса** → подсказка исчезает

---

## 🎨 Стиль Dark Souls 3:

- **Подсказка [Y]**: Внизу экрана, чёрная полупрозрачная, 250×40
- **Окно текста**: Внизу экрана, тёмное, 600×200
- **Заголовок**: Золотистый (0.95, 0.85, 0.6)
- **Текст**: Светло-серый (0.85, 0.85, 0.85)
- **Анимация**: Плавное появление/исчезновение

## 🔧 Настройка параметров:

### Signpost (инспектор):
| Параметр | Описание | Рекомендация |
|----------|----------|--------------|
| `Sign Title` | Заголовок окна | `"Древняя надпись"` |
| `Sign Text` | Текст подсказки | Многострочный текст |
| `Interaction Radius` | Радиус активации | `2-3` для мелких объектов |
| `Destroy After Reading` | Удалить после чтения | `❌` для многоразовых |

### InteractionUI (инспектор):
| Параметр | Описание | Рекомендация |
|----------|----------|--------------|
| `Fade In Speed` | Скорость появления | `5` |
| `Fade Out Speed` | Скорость исчезновения | `5` |

## 🐛 Отладка:

В консоли при работе системы появляются логи:
- `[Signpost] B нажат в InteractableObject!` - B зарегистрирован
- `[Signpost] OnClosePopup() вызван!` - вызвано закрытие
- `[Signpost] ClosePopup() вызван!` - popup закрывается
- `[InteractionUI] HideTextPopup() вызван!` - UI скрывается

Если popup не закрывается - проверь консоль на эти сообщения!

##  Создание новых типов интерактивных объектов:

Чтобы создать новый тип (например, подбираемый предмет):

```csharp
using UnityEngine;

public class PickupItem : InteractableObject
{
    public string itemName = "Зелье";
    public int quantity = 1;

    public override void Interact()
    {
        isShowingPopup = true;
        
        // Логика подбора предмета
        Debug.Log($"Подобрано: {itemName} x{quantity}");
        
        // Показать сообщение
        if (InteractionManager.Instance != null)
        {
            InteractionManager.Instance.ShowTextPopup("Подобрано!", $"Вы получили {itemName} x{quantity}");
            InteractionManager.Instance.HideInteractionPrompt();
        }
        
        // Уничтожить объект
        Destroy(gameObject, 2f);
    }

    protected override void OnClosePopup()
    {
        isShowingPopup = false;
        if (InteractionManager.Instance != null)
            InteractionManager.Instance.HideTextPopup();
    }
}
```
