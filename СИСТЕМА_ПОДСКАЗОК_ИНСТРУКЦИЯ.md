# 🎮 Простая система подсказок

## 📋 Что осталось:

| Файл | Назначение |
|------|-----------|
| `InteractableObject.cs` | Базовый класс (радиус, кнопки Y/B) |
| `Signpost.cs` | Скрипт для подсказок |

## 🛠 Настройка в Unity:

### 1️⃣ Создай UI (сам):

**Подсказка [Y]:**
- Создай Canvas → Panel → назови "PromptPanel"
- Добавь текст "[Y] Прочитать"
- **Скрой** по умолчанию (SetActive = false)

**Окно текста:**
- Создай Panel → назови "PopupPanel"  
- Добавь текст подсказки
- **Скрой** по умолчанию (SetActive = false)

### 2️⃣ Создай подсказку:

```
1. GameObject → 3D Object (Cube/модель)
2. Add Component → Signpost
3. В инспекторе перетащи:
   - Prompt Panel: твой PromptPanel
   - Popup Panel: твой PopupPanel
4. Настрой Interaction Radius: 2-3
```

### 3️⃣ Как работает:

| Действие | Что происходит |
|----------|----------------|
| **Подойти** | Появляется PromptPanel |
| **Нажать Y** | PromptPanel скрывается, PopupPanel появляется |
| **Нажать B** | PopupPanel скрывается, PromptPanel появляется |
| **Отойти** | Всё скрывается |
| **Подойти снова** | Снова появляется PromptPanel |

### 4️⃣ Несколько подсказок:

Просто создай несколько объектов с `Signpost` - каждый работает независимо!
