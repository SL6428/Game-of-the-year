# 🎮 Система подсказок в стиле Dark Souls 3

## 📋 Что это такое?

Система интерактивных подсказок как в **Dark Souls 3**:
- Подходишь к объекту → внизу экрана появляется `[Y] Прочитать`
- Нажимаешь Y → красивое полупрозрачное окно с текстом
- Нажимаешь B → закрываешь окно

---

## 🛠 Настройка в Unity (5 минут)

### Шаг 1: Создать менеджер на сцене

1. В Unity создай пустой объект: **GameObject → Create Empty**
2. Переименуй в **`InteractionManager`**
3. Добавь компонент: **Add Component → `InteractionManager`**

**ВСЁ!** UI создастся автоматически при запуске!

---

### Шаг 2: Создать подсказку на сцене

1. Создай 3D объект который будет подсказкой:
   - **GameObject → 3D Object → Cube** (или любой другой)
   - Или используй свою модель (доска, записка, труп)

2. Переименуй в **`Signpost`** (или как хочешь)

3. Добавь компонент: **Add Component → `Signpost`**

4. Настрой в инспекторе:
   ```
   Signpost (Script)
   ├─ Interaction Prompt: "[Y] Прочитать"
   ├─ Interaction Radius: 2
   ├─ Player: [перетащи игрока]
   ├─ Sign Title: "Заброшенная записка"
   ├─ Sign Text: "Тот кто придёт сюда после меня..."
   └─ Destroy After Reading: ❌ (или ✅ если одноразовая)
   ```

5. **Готово!** Запускай сцену и проверяй!

---

### Шаг 3: Настроить игрока (если ещё не настроен)

1. Выбери объект игрока
2. Убедись что у него **тег "Player"** (важно!)
3. (Опционально) Добавь компонент `PlayerInteraction` для проверки

---

## 🎨 Настройка внешнего вида UI

### В инспекторе InteractionUI:

**Prompt UI** (подсказка внизу экрана):
- `Prompt Panel` - панель подсказки
- `Prompt Text` - текст "[Y] Прочитать"

**Text Popup UI** (окно с текстом):
- `Popup Panel` - главная панель окна
- `Popup Title` - заголовок (золотистый)
- `Popup Content` - текст подсказки
- `Popup Close Hint` - "[B] Закрыть"

**Animation Settings**:
- `Fade In Speed` - скорость появления (5 = быстро)
- `Fade Out Speed` - скорость исчезновения

## 📝 Создание разных типов подсказок

### Тип 1: Обычная записка/доска

```csharp
// Создай новый скрипт ReadableNote.cs
public class ReadableNote : InteractableObject
{
    [TextArea(5, 10)]
    public string noteTitle = "Записка";
    
    [TextArea(10, 20)]
    public string noteText = "Текст записки...";
    
    public override void Interact()
    {
        isShowingPopup = true;
        InteractionManager.Instance.ShowTextPopup(noteTitle, noteText);
        InteractionManager.Instance.HideInteractionPrompt();
    }
    
    protected override void OnPlayerExitRange()
    {
        base.OnPlayerExitRange();
        if (isShowingPopup) ClosePopup();
    }
    
    public void ClosePopup()
    {
        isShowingPopup = false;
        InteractionManager.Instance.HideTextPopup();
    }
    
    void Update()
    {
        base.Update();
        if (isShowingPopup && Input.GetKeyDown(KeyCode.B))
            ClosePopup();
    }
}
```

### Тип 2: Предмет который можно подобрать

```csharp
public class PickupItem : InteractableObject
{
    public string itemName = "Зелье лечения";
    public int quantity = 1;
    
    public override void Interact()
    {
        // Подбираем предмет
        Debug.Log($"Подобрано: {itemName} x{quantity}");
        
        // Добавляем в инвентарь (потом)
        // Inventory.Add(itemName, quantity);
        
        // Показываем сообщение
        InteractionManager.Instance.ShowTextPopup("Подобрано!", $"Вы получили {itemName} x{quantity}");
        InteractionManager.Instance.HideInteractionPrompt();
        
        // Уничтожаем объект
        Destroy(gameObject, 2f);
    }
}
```

## 🎯 Как это работает (архитектура)

```
InteractionManager (Singleton)
    ├─ InteractionUI (управляет Canvas/UI)
    │   ├─ Prompt Panel ([Y] внизу экрана)
    │   └─ Text Popup (окно с текстом)
    │
    └─ Signpost (на сцене)
        ├─ Проверяет дистанцию до игрока
        ├─ Показывает prompt когда игрок близко
        ├─ Открывает popup при нажатии Y
        └─ Закрывает при нажатии B или выходе из радиуса
```

## ⚙️ Параметры в инспекторе

### Signpost:
| Параметр | Описание | Рекомендация |
|----------|----------|--------------|
| `Interaction Prompt` | Текст кнопки | `"[Y] Прочитать"` |
| `Interaction Radius` | Радиус активации | `2-3` для мелких, `4-5` для крупных |
| `Player` | Ссылка на игрока | Перетащи игрока |
| `Sign Title` | Заголовок окна | `"Древняя надпись"` |
| `Sign Text` | Текст подсказки | Многострочный текст |
| `Destroy After Reading` | Удалить после чтения | `❌` для многоразовых |

## 🎨 Стилизация под Dark Souls 3

Цвета по умолчанию (уже настроены):
- **Фон окна**: Тёмный `(0.05, 0.05, 0.05, 0.85)`
- **Заголовок**: Золотистый `(0.95, 0.85, 0.6)`
- **Текст**: Светло-серый `(0.85, 0.85, 0.85)`
- **Подсказка**: Серый `(0.6, 0.6, 0.6)`

Шрифты (если хочешь изменить):
- Создай **TMP Font Asset** в Project окне
- Назначь его на TextMeshProUGUI компоненты

## 🐛 Проблемы и решения

| Проблема | Решение |
|----------|---------|
| Подсказка не появляется | Проверь тег игрока = "Player" |
| Текст не показывается | Проверь что InteractionManager на сцене |
| Не закрывается окно | Нажми B или выйди из радиуса |
| UI не создаётся | Проверь что есть Canvas на сцене |
| Текст не читается | Измени размер шрифта в InteractionUISetup.cs |
