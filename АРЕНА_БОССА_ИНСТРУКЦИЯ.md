# 🏰 Арена босса - Инструкция

## 🎯 Как работает

```
1. Стены ВЫКЛЮЧЕНЫ по умолчанию (игрок может ходить свободно)
2. Игрок входит в триггер-зону → стены ВКЛЮЧАЮТСЯ (игрок заперт)
3. Бой с боссом
4. Босс умирает (Health.IsDead = true) → стены ВЫКЛЮЧАЮТСЯ (игрок свободен)
```

---

## 🛠 Настройка в Unity

### 1️⃣ Создать арену

```
1. GameObject → Create Empty → "BossArena"
2. Поставь в центр арены (где будет босс)
3. Add Component → BossArena
```

### 2️⃣ Настроить параметры

```
[Настройки арены]
- Arena Size: (20, 20)      ← Размер арены
- Wall Height: 5            ← Высота стен
- Wall Thickness: 0.5       ← Толщина стен

[Триггер-зона]
- Trigger Radius: 15        ← На каком расстоянии срабатывает

[Босс]
- Boss: перетащи босса
- Boss Health: оставь пустым (найдёт автоматически через Health компонент)

[Настройки]
- Show In Editor: ❌        ← Скрыть стены
- Show Trigger: ✅          ← Показать триггер (голубой круг)
```

### 3️⃣ Важно!

```
✅ У игрока ДОЛЖЕН быть тег "Player"
✅ У босса ДОЛЖЕН быть компонент Health (он уже есть!)
✅ Арена должна быть в центре между игроком и боссом
✅ Триггер создаётся ПРЯМО на объекте BossArena (не отдельный объект!)
```

---

## 🎮 Как использовать

### Автоматически:
```
1. Игрок подходит к арене (в радиусе Trigger Radius)
2. Стены включаются автоматически
3. Игрок не может уйти пока босс жив
4. Босс умирает → стены выключаются
```

### Вручную (из другого скрипта):
```csharp
BossArena arena = FindObjectOfType<BossArena>();

// Включить стены
arena.ActivateArena();

// Выключить стены
arena.DeactivateArena();
```

---

## 📐 Подбор размеров

### Триггер-радиус:
```
Trigger Radius должен быть БОЛЬШЕ чем половина Arena Size

Пример:
- Arena Size: (20, 20)
- Trigger Radius: 15 (должно быть > 10)
```

### Почему так?
```
Trigger Radius = 15
    ↓
    ⭕ Триггер (сфера радиусом 15)
    ↓
Игрок входит в сферу → стены включаются
```

## ❌ Исправленные проблемы

### Проблема: Триггер не срабатывал

**Было:**
```
❌ Триггер создавался как отдельный объект "ArenaTrigger"
❌ Игрок входил в триггер но OnTriggerEnter не вызывался
❌ Стены оставались выключенными
```

**Стало:**
```
✅ Триггер создаётся ПРЯМО на объекте BossArena
✅ SphereCollider.isTrigger = true
✅ Rigidbody.isKinematic = true
✅ OnTriggerEnter срабатывает когда игрок входит
```

## 🐛 Troubleshooting

### ❌ Стены не включаются

**Проверь:**
```
1. У игрока есть тег "Player"?
2. Игрок вошёл в голубой круг (триггер)?
3. Trigger Radius достаточно большой?
```

### ❌ Стены не выключаются после смерти босса

**Проверь:**
```
1. У босса есть компонент Health? (должен быть!)
2. Boss назначен в инспекторе?
3. Смотри консоль: "[BossArena] Найден Health у босса..."
```

### ❌ Ошибка "BossHealth could not be found"

**Решение:**
```
✅ ИСПРАВЛЕНО! Теперь используется Health (а не BossHealth)
Обнови скрипт BossArena.cs
```

### ❌ Игрок не может войти на арену

**Причина:** Триггер слишком маленький

**Решение:**
```
Увеличь Trigger Radius (15 → 20 → 25)
```

### ❌ Игрок застревает в стенах

**Причина:** Стены слишком близко

**Решение:**
```
Увеличь Arena Size (20 → 25 → 30)
```

## 💡 Как работает проверка смерти

```csharp
// BossArena автоматически находит Health компонент у босса
Health bossHealth = boss.GetComponent<Health>();

// И проверяет поле IsDead
if (bossHealth.IsDead) {
    // Босс мёртв → выключаем стены
    DeactivateArena();
}
```

**Health компонент уже есть у босса!** (EnemyAI требует Health через [RequireComponent])
