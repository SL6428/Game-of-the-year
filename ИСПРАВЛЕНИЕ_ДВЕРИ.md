# 🔧 Исправление: Дверь не открывается

## ❌ Что было не так

В предыдущей версии `Door.cs` был критический баг:

```csharp
void Update()
{
    // Анимация открытия
    if (isAnimating) { ... return; }

    // ❌ ЭТОТ БЛОК БЛОКИРОВАЛ ВСЮ ЛОГИКУ!
    if (isMessageShowing)
    {
        messageHideTimer -= Time.deltaTime;
        ...
        return; // ← Вот тут была проблема!
    }

    // Проверка расстояния (НИКОГДА НЕ РАБОТАЛА пока isMessageShowing = true)
    if (GameManager.Instance == null || ...) return;
    ...
}
```

### Проблема:
- Когда показывалось сообщение "Нет ключа", блок `if (isMessageShowing)` делал `return`
- Это **блокировало проверку расстояния** до игрока
- Дверь переставала реагировать на игрока пока сообщение не исчезнет
- Для двери **без ключа** (`requiresKey = false`) это тоже создавало проблемы

### Дополнительная проблема:
- Код зависел от `GameManager.Instance.Player` который мог быть `null`
- Не было запасного варианта поиска игрока по тегу

---

## ✅ Что исправлено

### 1️⃣ Разделена логика обновления

```csharp
void Update()
{
    // Если дверь уже открыта - ничего не делаем
    if (isOpen) return;

    // Анимация открытия
    if (isAnimating) { UpdateAnimation(); return; }

    // ✅ Обновляем ссылку на игрока (не блокирует!)
    UpdatePlayerReference();

    // ✅ Обновляем таймер сообщения (без return!)
    if (isMessageShowing)
    {
        messageHideTimer -= Time.deltaTime;
        if (messageHideTimer <= 0f)
        {
            isMessageShowing = false;
            if (noKeyMessagePanel != null)
                noKeyMessagePanel.SetActive(false);
        }
    }

    // ✅ Проверка расстояния РАБОТАЕТ ВСЕГДА
    if (playerTransform == null) return;
    ...
}
```

### 2️⃣ Добавлен UpdatePlayerReference()

```csharp
void UpdatePlayerReference()
{
    if (playerTransform == null)
    {
        // ✅ Сначала пробуем через GameManager
        if (GameManager.Instance != null && GameManager.Instance.Player != null)
        {
            playerTransform = GameManager.Instance.Player;
        }
        // ✅ Если нет - ищем по тегу (запасной вариант!)
        else
        {
            GameObject playerObj = GameObject.FindGameObjectWithTag("Player");
            if (playerObj != null)
            {
                playerTransform = playerObj.transform;
            }
        }
    }
}
```

### 3️⃣ Убрана блокировка isOpen в начале

```csharp
void Update()
{
    // ✅ Теперь проверка в самом начале
    if (isOpen) return;
    ...
}
```

### 4️⃣ Улучшена OpenDoor()

```csharp
void OpenDoor()
{
    isOpen = true;
    isAnimating = true;
    animationTimer = 0f;

    // Скрываем подсказку
    if (promptPanel != null)
        promptPanel.SetActive(false);

    // ✅ Скрываем сообщение если оно показано
    if (isMessageShowing)
    {
        isMessageShowing = false;
        if (noKeyMessagePanel != null)
            noKeyMessagePanel.SetActive(false);
    }

    // Звук
    if (openSound != null)
        openSound.Play();
}
```

## 🐛 Если всё ещё не работает

### Проверь консоль Unity:
```
- Есть ли ошибки?
- Есть ли предупреждения "[Door] Не назначен noKeyMessagePanel"?
- Есть ли "[GameManager] Не найден игрок"?
```

### Проверь настройки двери:
```
1. Выбери дверь в сцене
2. Проверь что Requires Key = ❌ (если дверь без ключа)
3. Проверь что Interaction Radius = 2 или больше
4. Проверь что Prompt Panel назначен
```

### Проверь игрока:
```
1. Выбери игрока в сцене
2. Убедись что у него есть тег "Player"
3. Убедись что игрок активен (галочка включена)
```

### Добавь отладку:
```
Вставь Debug.Log в Update() двери:

void Update()
{
    Debug.Log($"[Door] Update: isOpen={isOpen}, isAnimating={isAnimating}");
    ...
}
```
