## 🚀 СЛЕДУЮЩИЕ ШАГИ (обязательно!)

### 1. Открыть Unity проект
```
Запустите Unity Hub → откройте проект Game-of-the-year
```

### 2. Назначить тег "Player" игроку
```
1. Откройте сцену: Assets/Scenes/Gail map (first map).unity
2. Найдите объект игрока в иерархии
3. В инспекторе нажмите на тег
4. Выберите "Player"
5. Сохраните сцену (Ctrl+S)
```

### 3. Добавить компонент PlayerRegeneration
```
1. Выберите объект игрока
2. Add Component → PlayerRegeneration
3. Настройте (по умолчанию уже верно):
   - Max Charges: 5
   - Recharge Time: 15
   - Heal Amount: 30
```

### 4. Запустить валидатор
```
В меню Unity: Tools → Combat System Validator
Нажмите "▶️ Run All Checks"
Исправьте все ошибки (помечены красным)
```

### 5. Протестировать игру
```
1. Нажмите Play
2. Проверьте:
   - □ Движение игрока (WASD)
   - □ Атака (ЛКМ) → урон наносится врагам
   - □ Лечение (H) → HP восстанавливается
   - □ Враги атакуют → игрок получает урон
   - □ Смерть врагов при 0 HP
```

## 🔧 Настройки компонентов

### PlayerRegeneration (на игроке)
```
Max Charges: 5          // Максимум зарядов
Recharge Time: 15       // Секунд на перезарядку
Heal Amount: 30         // Здоровья за использование
```

### EnemyAI (на каждом враге)
```
Damage: 10              // Урон игроку
Attack Range: 2         // Дистанция атаки
Attack Cooldown: 1.5    // Секунд между атаками
Detection Range: 10     // Дистанция обнаружения
```

### Weapon (на оружии игрока)
```
Damage: 20              // Урон врагу
Enemy Layers: Enemy     // Слой врагов
```

## 🐛 Если что-то не работает

### Ошибка компиляции
```
Решение: Удалите Library и obj папки, переоткройте проект
```

### Игрок не получает урон от врагов
```
Проверки:
1. У игрока есть тег "Player"?
2. У врага в EnemyAI назначен игрок?
3. У игрока есть компонент Health?
```

### Лечение не работает
```
Проверки:
1. Есть ли компонент PlayerRegeneration на игроке?
2. Есть ли доступные заряды?
3. HP не полное?
```

### Оружие не наносит урон
```
Проверки:
1. У врага есть тег "Enemy"?
2. У врага есть Health?
3. Коллайдер оружия — триггер?
4. Анимация вызывает EnableHitbox?
```
