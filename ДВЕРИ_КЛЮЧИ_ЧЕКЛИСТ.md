# ✅ Чеклист настройки дверей и ключей

## 1️⃣ Проверка GameManager

- [ ] На сцене есть объект "GameManager"
- [ ] У объекта есть компонент `GameManager`
- [ ] В поле **Player** назначен игрок (или у игрока есть тег "Player")
- [ ] GameManager не уничтожается между сценами

---

## 2️⃣ Проверка UI Canvas

- [ ] На сцене есть Canvas
- [ ] Создана панель **DoorPromptPanel** (скрыта по умолчанию)
- [ ] Создана панель **DoorNoKeyMessagePanel** (скрыта по умолчанию)
- [ ] Создана панель **KeyPromptPanel** (скрыта по умолчанию)
- [ ] Создана панель **KeyPickupMessagePanel** (скрыта по умолчанию)
- [ ] У каждой панели есть дочерний объект Text
- [ ] Все панели отключены (галочка снята) в редакторе

---

## 3️⃣ Проверка KeyPickup

- [ ] У объекта ключа есть компонент `KeyPickup`
- [ ] **Key Name** заполнен (например "UpperHallKey")
- [ ] **Pickup Radius** установлен (рекомендуется 2)
- [ ] **Prompt Panel** назначен (KeyPromptPanel)
- [ ] **Pickup Message Panel** назначен (KeyPickupMessagePanel)
- [ ] **Pickup Message Text** назначен (Text внутри KeyPickupMessagePanel)
- [ ] **Message Duration** установлен (рекомендуется 2)
- [ ] **Destroy After Pickup** установлен по желанию

---

## 4️⃣ Проверка Door (без ключа)

- [ ] У объекта двери есть компонент `Door`
- [ ] **Requires Key** = ❌ (выключено)
- [ ] **Interaction Radius** установлен (рекомендуется 2)
- [ ] **Prompt Panel** назначен (DoorPromptPanel)
- [ ] **Rotation Axis** установлен (обычно (0, 1, 0))
- [ ] **Open Angle** установлен (рекомендуется 90)
- [ ] **Open Duration** установлен (рекомендуется 1.5)
- [ ] Дверь в закрытом положении перед запуском

---

## 5️⃣ Проверка Door (с ключом)

- [ ] У объекта двери есть компонент `Door`
- [ ] **Requires Key** = ✅ (включено)
- [ ] **Required Key Name** заполнен (должен совпадать с KeyPickup.Key Name!)
- [ ] **Interaction Radius** установлен (рекомендуется 2)
- [ ] **Prompt Panel** назначен (DoorPromptPanel)
- [ ] **No Key Message Panel** назначен (DoorNoKeyMessagePanel)
- [ ] **No Key Message Text** назначен (Text внутри DoorNoKeyMessagePanel)
- [ ] **Message Duration** установлен (рекомендуется 2)
- [ ] **Rotation Axis** установлен (обычно (0, 1, 0))
- [ ] **Open Angle** установлен (рекомендуется 90)
- [ ] **Open Duration** установлен (рекомендуется 1.5)
- [ ] Дверь в закрытом положении перед запуском

---

## 6️⃣ Проверка SimpleInventory

- [ ] SimpleInventory создаётся автоматически (не нужно создавать вручную)
- [ ] При запуске в консоли появится "[Door] Создан SimpleInventory" или "[KeyPickup] Создан SimpleInventory"

---

## 7️⃣ Проверка тега игрока

- [ ] У объекта игрока есть тег **"Player"**
- [ ] Тег написан точно (case-sensitive!)
- [ ] GameManager может найти игрока по тегу

---

## 8️⃣ Финальная проверка

Запустите сцену и проверьте:

- [ ] В консоли нет ошибок (кроме предупреждений)
- [ ] При приближении к ключу появляется "[E] Подобрать"
- [ ] При нажатии E у ключа появляется "🔑 Подобран ключ"
- [ ] Ключ исчезает после подбора
- [ ] "[E] Подобрать" исчезает после подбора
- [ ] При приближении к двери появляется "[E] Открыть"
- [ ] Если нет ключа и Requires Key = ✅, появляется "Требуется ключ"
- [ ] Если есть ключ, дверь открывается
- [ ] Если Requires Key = ❌, дверь открывается сразу
- [ ] Все сообщения исчезают через 2 секунды

---

## ❌ Если есть проблемы

### "Player not found" в консоли:
```
→ Добавь тег "Player" объекту игрока
→ Или перетащи игрока в GameManager.Player
```

### UI панели не появляются:
```
→ Проверь что панели назначены в инспекторе
→ Проверь что панели скрыты по умолчанию (галочка)
→ Проверь что Text внутри панелей назначен
```

### Дверь не открывается:
```
→ Проверь Requires Key и Required Key Name
→ Убедись что ключ подобран (смотри консоль)
→ Проверь что дверь ещё не открыта
```

### Ключ не подбирается:
```
→ Проверь что KeyPickup компонент добавлен
→ Проверь что isPickedUp = false (ключ ещё не подобран)
→ Проверь что SimpleInventory создан (смотри консоль)
```

### Сообщения не исчезают:
```
→ Проверь что Message Duration > 0
→ Проверь что нет ошибок в консоли
```
